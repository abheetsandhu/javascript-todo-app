# JavaScript To-Do App

## Overview
This is a simple front-end To-Do List application built with HTML, CSS, and JavaScript. Users can add tasks, mark them as complete, and delete them.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Basic input validation
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Skills Demonstrated
- DOM manipulation
- Event handling
- Front-end development basics
- User interface design
- Basic input validation

## How to Run
1. Download the project files.
2. Open `index.html` in a web browser.
3. Add, complete, and delete tasks.

## Why I Built This
I built this project to practice basic JavaScript, front-end development, DOM manipulation, and user interaction.
